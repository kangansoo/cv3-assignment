# PRD

## 개요

이 프로젝트는 방송 데이터 목록을 유형별로 조회할 수 있는 간단한 웹 애플리케이션입니다.

사용자는 라이브 방송과 홈쇼핑 유형을 전환하며, 각 유형에 해당하는 방송 목록을 테이블로 확인할 수 있습니다.

## 목표

- 방송 데이터를 유형별로 조회할 수 있다.
- 각 유형별 최대 10개의 방송 정보를 표시한다.
- 원본 데이터의 표시값을 최대한 동일하게 유지한다.
- 사용자가 별도의 설정 없이 로컬에서 실행할 수 있다.

## 주요 기능

### 방송 유형 전환

사용자는 다음 두 유형 중 하나를 선택할 수 있습니다.

- 라이브 방송
- 홈쇼핑

선택한 유형에 따라 테이블 데이터가 변경됩니다.

### 방송 목록 테이블

테이블은 다음 정보를 표시합니다.

- 방송정보
- 분류
- 방송시간
- 조회수/시청률
- 판매량
- 매출액
- 상품수

각 유형별로 최대 10개의 항목만 표시합니다.

## 비기능 요구사항

- UI는 간결하게 구성한다.
- 데이터 요청 및 정규화 로직은 UI와 분리한다.
- 불필요한 의존성은 추가하지 않는다.
- README만 보고 실행할 수 있어야 한다.
